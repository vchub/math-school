# Тема: Вынесение общего множителя за скобки

## Описание темы
Вынесение общего множителя за скобки — это важный алгебраический приём, обратный раскрытию скобок. Он позволяет представить сумму (или разность) в виде произведения, что упрощает дальнейшие преобразования выражений, решение уравнений и работу с многочленами.

**Обязательно посмотрите видеоурок:** https://youtu.be/ADJEdClfdes?si=Ho1AMRdMrKxFzwWE

## Основная идея и алгоритм

**Основная идея:** Из суммы «вытаскиваем» одинаковый множитель, который присутствует в каждом слагаемом, и записываем выражение как произведение.

**Алгоритм:**
1. Найдите **общий множитель** всех слагаемых (число, переменную или их комбинацию). Выбирайте **наибольший** возможный общий множитель (НОД для чисел, наименьшую степень для переменных).
2. Вынесите этот множитель за скобку.
3. В скобках запишите результаты деления каждого слагаемого на вынесенный множитель.
4. Если выносите отрицательный множитель — знаки внутри скобок меняются на противоположные.

**Важные моменты:**
- Когда слагаемое делится полностью (например, 9 делим на 9), в скобках остаётся **1**, а не 0.
- Общий множитель может быть целым выражением в скобках (например, (x + 2)).

## Примеры

### Пример 1 (числовой множитель)
$$12x - 18y$$
- НОД 12 и 18 — 6  
- $12x : 6 = 2x$, $-18y : 6 = -3y$  
- Ответ: $6(2x - 3y)$

### Пример 2 (с единицей)
$$9a + 9$$
- Общий множитель — 9  
- $9a : 9 = a$, $9 : 9 = 1$  
- Ответ: $9(a + 1)$

### Пример 3 (переменная)
$$a^3 - 5a^2$$
- Общая переменная в наименьшей степени — $a^2$  
- $a^3 : a^2 = a$, $-5a^2 : a^2 = -5$  
- Ответ: $a^2(a - 5)$

### Пример 4 (отрицательный множитель)
$$-7x - 14y$$
- Выносим -7  
- Знаки меняются: $-7x : (-7) = x$, $-14y : (-7) = 2y$  
- Ответ: $-7(x + 2y)$

### Пример 5 (общая скобка)
$$a(b+1) - 2(b+1)$$
- Общий множитель — $$(b+1)$$  
- Ответ: $$(b+1)(a - 2)$$

## Задания для самостоятельного решения

Задания отсортированы по возрастанию сложности.

### Базовый уровень

1. Вынесите общий множитель: $$5a + 10b$$
2. Разложите на множители: $$12y - 12$$
3. Вынесите общий множитель: $$7x - 21$$
4. Представьте в виде произведения: $$15 + 5k$$
5. Вынесите общий множитель: $$-4m - 16n$$
6. Разложите на множители: $$8c + c$$
7. Вынесите общий множитель: $$18 - 27p$$
8. Представьте в виде произведения: $$33a - 11b$$
9. Разложите на множители: $$-9x + 27$$
10. Вынесите общий множитель: $$a - 5a$$

### Продвинутый уровень

1. Вынесите общий множитель: $$a^2 - 6a$$
2. Разложите на множители: $$8xy + 12y$$
3. Вынесите общий множитель: $$c^3 + c^5$$
4. Какой вид примет выражение $$x(y-z) + 5(y-z)$$ после вынесения общего множителя?
5. Разложите на множители: $$6a^3 - 3a^2 + 12a$$
6. Представьте выражение $$-x^4 - 2x^3 - 3x^2$$ в виде произведения.
7. Вынесите общий множитель: $$5(a+b) - c(a+b)$$
8. Разложите на множители: $$m^2n - mn^2$$
9. Упростите выражение $$3x(x-2) - 5(2-x)$$
10. Какой множитель является наибольшим общим для выражения $$16z^2w - 24zw^2$$?
